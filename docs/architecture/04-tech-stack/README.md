# Technology Stack

## Core

- **Runtime**: Node.js (Development)
- **Framework**: React 18
- **Build Tool**: Vite (Fast HMR and bundling)
- **Language**: JavaScript (ES Modules)

## State Management

- **Global**: React Context API
- **Local/Complex**: `useReducer`

## Key Dependencies

| Package            | Purpose                                          |
| ------------------ | ------------------------------------------------ |
| `react-router-dom` | Client-side routing                              |
| `@dnd-kit/core`    | Drag and drop primitives                         |
| `fuse.js`          | Fuzzy search (likely for Pokedex or Item search) |
| `lucide-react`     | Icons                                            |
| `json-server`      | Mock REST API for development                    |

## Development Tools

- **Linter**: ESLint with React guidelines
- **Formatter**: Prettier
- **Testing**: Vitest + React Testing Library
