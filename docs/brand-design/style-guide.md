# Brand Style Guide

## Color Palette

The color scheme is inspired by the classic Generation 1 Pokemon games, using vibrant primary colors balanced with clean neutrals.

### Primary Colors
| Color Name | Hex Code | Usage |
|:---:|:---:|:---|
| **Pokemon Yellow** | `#ffcb05` | Primary actions, highlights, branding elements |
| **Pokemon Blue** | `#2a75bb` | Headers, secondary buttons, info states |
| **Pokemon Red** | `#ff0000` | Accents, alerts, critical health indicators |

### Neutral Colors
| Color Name | Hex Code | Usage |
|:---:|:---:|:---|
| **Background** | `#f0f8ff` | App background (Light Alice Blue) |
| **Text Primary** | `#1a1a1a` | Main content text |
| **Text Secondary**| `#595959` | Subtitles, meta info |
| **Card Bg** | `#ffffff` | Component backgrounds |

### Status Colors
- **Health Good**: `#00ff00`
- **Health Warning**: `#ffaa00`
- **Health Critical**: `#ff0000`

---

## Typography

### Headers
- **Font Family**: `Luckiest Guy`, cursive
- **Usage**: Main page titles, modal headers, gamified text elements.
- **Styling**: Often used with text-shadow or stroke to mimic the Pokemon logo style.

### Body Text
- **Font Family**: `Inter`, sans-serif
- **Usage**: readable UI text, descriptions, buttons, tooltips.

---

## UI Elements

### Buttons
- **Primary**: Yellow background (`#ffcb05`) with dark text. rounded corners.
- **Secondary**: Blue background (`#2a75bb`) with white text.

### Cards
- White background (`#ffffff`) with subtle shadow.
- Rounded corners (`border-radius: 8px` or similar).

### Gradients
- The main background uses a radial gradient pattern: `radial-gradient(#e0e0e0 1px, transparent 1px)` with `20px` spacing to create a subtle grid effect.
