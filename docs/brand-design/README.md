# Brand Design

This directory contains assets and guidelines for the visual identity of Pokemon Felix.

## Contents

- **[Style Guide](./style-guide.md)**: Comprehensive guide to colors, fonts, and UI components.
- **Assets**: (Planned location for logos and icons)
  - `logo.png`
  - `logo_with_name.png`

## Vision

The visual design aims to be:

1.  **Nostalgic**: Evoking the feel of early handheld games.
2.  **Clean**: Modern UI practices (whitespace, clear typography) applied to retro aesthetics.
3.  **Playful**: Using "Luckiest Guy" font and vibrant colors to maintain a fun atmosphere.
