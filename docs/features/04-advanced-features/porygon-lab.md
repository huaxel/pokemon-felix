# Porygon Lab

## Overview
An educational mini-game designed to teach basic programming logic (Sequencing, Loops) to children.

## Mechanics
- **Grid**: A 5x5 tile board.
- **Avatar**: Porygon (Digital Pokemon).
- **Goal**: Reach the endpoint (Flag) by queuing commands.

## Commands
1. **Forward**: Move 1 tile in facing direction.
2. **Left/Right**: Rotate 90 degrees.
3. **Jump**: Move 2 tiles (Hop over obstacles).

## Levels
Levels introduce complexity progressively:
1. **Hola Mundo**: Straight line.
2. **El Giro**: Introduction to turning.
3. **El Salto**: Introduction to JUMP to cross gaps.
